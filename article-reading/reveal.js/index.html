Deux attitudes face à la création

Spiritualiste

  • Artiste = réceptacle sacré
  • Méfiance envers la raison
  • L'outil est un écran

Matérialiste

  • Artiste = chercheur
  • Alliance avec la science
  • L'outil est une prothèse

Le défi de l'image mentale

  • La création part d'une vision interne floue.
  • Les limites humaines : fatigue, manque de temps, incohérences.
  • Le "replâtrage" : l'obligation de camoufler les erreurs en fin de travail.

L'ordinateur : "Machine à mieux voir"

"L'ordinateur ne sait ni rêver ni imaginer ni inventer."

Mais il permet de passer du flou subjectif au "sharp focus".

Ses rôles clés :

  • Contrôle : Gestion de milliers de points quantifiés.
  • Mémoire : Possibilité de revenir sur chaque étape.
  • Fidélité : Matérialiser ce qu'il y a de plus profond en soi.

Le dialogue Homme-Machine en pratique


								
								var w, h
								var cnv

								function setup() {
									w = 800
									h = 800
									cnv = createCanvas(w, h);
									centerCanvas();
									colorMode(HSB, 360, 100, 100, 250);
									noLoop()
								}

								function centerCanvas() {
									var x = (windowWidth - 800) / 2;
									var y = (windowHeight - 800) / 2;
									cnv.position(x, y);
								}

								function draw() {
									background(0,0,0)
									fill(230,100,100) // couleur bleu
									noStroke()
									let noir=false
									let cx=w*0.5
									let cy=h*0.5
									let vera=random(42,300)
									let molnar=27
									let pas=29
									for(vera=300;vera>42;vera-=pas){
										noir?fill(0,0,0):fill(230,100,100)
									quad(cx-vera+random(-molnar,molnar),cy-vera+random(-molnar,molnar),
										cx+vera+random(-molnar,molnar),cy-vera+random(-molnar,molnar),
										cx+vera+random(-molnar,molnar),cy+vera+random(-molnar,molnar),
										cx-vera+random(-molnar,molnar),cy+vera+random(-molnar,molnar)
									)
									noir=!noir
								}
								}

L'artiste définit les règles et l'ordinateur gère les milliers de points.

Résultat en temps réel : Une "image puissance image"[cite: 128].

  • Réunir la micro-esthétique (création) et la macro-esthétique .
  • La recherche systématique dérange le milieu de l'art lucratif.
  • "La science de l'art est déjà née."
